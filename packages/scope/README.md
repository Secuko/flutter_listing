# Либа для базовых классов скоупов bloc_scope.dart
                                  scope.dart
                                  scope_access.dart
                                  stream_listener.dart

    Пакет используется для создания интерфейса взаимодействия с блоком на основе
инхерит виджета. Так же виджеты подписываются на изменение необходимого для него параметра,
без необходимости ребилдить весь экран, поэтому BlocBuilder НЕ используется.

final data = HomeScope.data(context, listen: true);
listen - параметр, отвечающий за необходимость подписывать виджет на изменение этого параметра

    BlocScope используется для доступа к блоку внутки скоупа.
Доступ к этой переменной внутри виджета скоупа:

static const BlocScope<ExampleEvent, ExampleState, ExampleBloc> _scope = BlocScope();

    Если необходимы какие-то параметры состояния, которые не в
State блока, то для этого стоит использовать виджет, наследующийся от виджета Scope.
Для доступа к такому виджету внутри него, необходимо использовать следующий код:

static const ScopeAccess<TestScope, _TestScopeDelegate> _scopeAccess = ScopeAccess();

Так же, если есть listenable переменные, то для ребилда их необходимо добавить в dependOnListenables делегата.

StreamListener необходимо использовать как аналог BlocListener, который должен
находиться в другой папке для облегчения понимания кода.
